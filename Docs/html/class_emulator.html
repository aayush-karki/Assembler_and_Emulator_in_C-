Detailed Description

This is a class for emulator.

Emulator class supports the emulation of VC1620 programs. It runs the trananslated assembly code.

Warning
Emulator does not checks if any memory address, which are in bounds, are valid access or not. This is done by assembler.
If only emulator is used, there are lot of errors which it does not accounts for.
